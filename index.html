프로젝트: AI 중고차 구매 가이드

문제 정의

중고차 '성능·상태 점검기록부' 해독의 어려움과 정보 비대칭으로 인한 소비자 불안 문제를 AI 기술로 해결하고자 했습니다.

아키텍처/설계

React 프론트엔드와 Python/Flask 백엔드로 구성된 AI 앱입니다.

  1. Frontend (React): 파일 업로드, 라이프스타일 입력, 결과 확인, 챗봇 등 UI 구현. `fetch`로 백엔드와 비동기 통신.
  2. Backend (Flask): 핵심 AI API 엔드포인트 3개(`/api/upload`, `/api/recommend`, `/api/ask-report`) 구현.
  3. AI/OCR 연동: Google Document AI (OCR)로 텍스트를 추출하고, Gemini AI로 JSON 리포트 생성, 추천, Q&A 수행.

핵심 기능 1: Google AI (Document AI + Gemini)를 활용한 성능기록부 자동 분석

사용자가 성능기록부(PDF/이미지)를 업로드하면, 백엔드에서 OCR로 텍스트를 추출하고 OCR 오류를 보정합니다. 이후 Gemini AI가 텍스트를 분석하여, 사고 이력, 누유, 적정 시세, 위험도 점수(0-100)가 포함된 상세 JSON 리포트를 생성합니다.

핵심 기능 2: Gemini AI 기반 맞춤형 중고차 추천 및 리포트 Q&A 챗봇

  1. AI 맞춤 추천: 라이프스타일(예: '30대, 주말 캠핑, 2천만 원')을 입력하면 AI가 예산, 용도 등을 고려해 최적 중고차 3가지를 추천 이유, 시세와 함께 JSON으로 반환합니다.
  2. 리포트 Q&A 챗봇: 생성된 리포트와 사용자 질문(예: "누유 심각한가요?")을 AI에 전달하여, 리포트 내용에 기반한 정확한 답변을 실시간 제공합니다.

  • OCR 분석, 맞춤형 추천, Q&A 챗봇이라는 3가지 핵심 AI 기능을 Gemini AI로 성공적으로 통합
